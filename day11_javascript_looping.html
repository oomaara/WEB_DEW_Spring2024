JavaScript - Looping and Switches

Looping

While Loop

While loops are used to execute a block of code multiple times as long as a condition is true

Example:

        
            let i = 0;
            while (i < 5){
                console.log(i);
                i++
            }

Output:

Do While Loop

Do while loops are used to execute a block of code at least once, and then repeat the loop as long as a condition is true

Example:

        
            let i = 0;
            do {
                console.log(i);
                i++
            } while (i < 5);

Output:

I call do while loops "Shoot first and ask questions later" loops because they check the condition AFTER one run of code inside the loop block

You can simulate do while loop with regular while loop using break

For Loop

For loops are used to execute a block of code a number of times

Example:

        
            for (let i = 0; i < 5; i++){
                console.log(i);
            }

Switches

Example:

        
            let day = new Date().getDay();
            switch (day){
                case 0: //in JS Sunday is 0
                    console.log("Sunday");
                    break;
                case 1:
                    console.log("Monday");
                    break;
                case 2:
                    console.log("Tuesday");
                    break; 
                case 3:
                    console.log("Wednesday");
                    break;
                case 4:
                    console.log("Thursday");
                    break;
                case 5:
                    console.log("Friday");
                    break;
                case 6:
                    console.log("Saturday");
                    break;
                default:
                    console.log("Unknown day");
            }

Output:

Switches are useful as a tool to replace multiple if else statements

In case of numeric values that start with 0 we could have used array and gotten its elements by index But in case of string values, we can't use array, so we use switches

Arrays will be discussed in another lecture
