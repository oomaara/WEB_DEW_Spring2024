Introduction to JavaScript

What is JavaScript

JavaScript is a programming language that enables you to create dynamically updating content, control multimedia, animate images, and much more. It is a client-side script language that runs in the browser

Data Types in JavaScript

There are 7 basic data types in JavaScript:

  • String - for text
  • Number - any numbers 1, 3,14, -356, 0, etc.
  • Boolean
  • Null
  • Undefined
  • Object
  • Symbol
